# wordpress-theme-mystyle
一款集成瀑布流、栅格布局的WordPress主题
## 支持的WordPress版本
* 支持wp3.5及以下的版本；
* 在高版本WordPress下安装使用可能造成部分功能失效；

## 主题功能特性
* 首次采用当下流行的瀑布流布局
* 多种不同category，可实现多种不同布局：瀑布流、多栏栅格布局、普通图文布局
* 后台内置多种功能，请安装后自行体验

## 关于使用
* 主题开源并可完全免费使用
* 请保留作者链接和版权信息
* 可自行二次开发

## 接受贡献
* 欢迎对主题源码进行纠错、改进、冗余代码清除
* 可以pull requests/issues的方式进行提交或发问
